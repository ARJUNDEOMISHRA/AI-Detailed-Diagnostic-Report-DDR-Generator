# AI-Detailed-Diagnostic-Report-DDR-Generator
# AI DDR Generator

This project builds an AI system that converts inspection and thermal reports into structured Detailed Diagnostic Reports (DDR).

## Features

- Structured JSON extraction
- Conflict detection
- Missing information handling
- Professional diagnostic report generation
- Generalizable pipeline

## Tech Stack

- Python
- Google Gemini AI
- Jupyter Notebook

## Project Architecture

inspection.txt + thermal.txt  
↓  
Structured JSON extraction  
↓  
DDR Report generation  
↓  
Save outputs  

## Output Files

- extracted_data.json
- final_ddr_report.txt

## How to Run

1. Open DDR_Generator_Final.ipynb
2. Add inspection.txt and thermal.txt
3. Run all cells
4. DDR report generated automatically
